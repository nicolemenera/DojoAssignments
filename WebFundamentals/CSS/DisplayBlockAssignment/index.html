Javascript Basics

JavaScript is a cross-platform, object oriented scripting language. Javascript is extremely popular for a variety of reasons. It is a small and lightweight laguage allowing maximum flexibility for developers to take it in a bunch of different directions. JavaScript lives inside a host environment (a web browse or Node server), it can be connected to the objects of these environments to provide programmatic control over them.
